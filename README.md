# Semester HQ

A calm, self-explanatory dashboard for your semester — courses, assignments, and exams with a "this week" view, a month calendar, and workload at a glance.

Also an installable mobile app: Semester HQ is a Progressive Web App, so you can add it to your home screen (iPhone: Share → Add to Home Screen; Android: Install) and use it fully offline.

Designed by Regan Hinton and built with an AI-accelerated workflow (Claude Code). Plain HTML/CSS/JavaScript, no build step. Data stays in your browser, with optional cross-device sync via a private GitHub gist.

**Live:** https://rlhinton04.github.io/semester-hq/
