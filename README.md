# Semester HQ

A calm, self-explanatory dashboard for your semester — courses, assignments, and exams with a "this week" view and workload at a glance.

Designed by Regan Hinton and built with an AI-accelerated workflow (Claude Code). Plain HTML/CSS/JavaScript, no build step, all data stays in your browser.

**Live:** https://rlhinton04.github.io/semester-hq/
